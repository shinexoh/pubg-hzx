# 画质侠

一款和平精英画质助手，是之前开发的 [Doge画质助手](https://github.com/shinexoh/dogehzzs) 升级版。仅支持安卓手机，没什么技术含量 :)

## 应用功能

* 一些简单的画质修改
* 通过购买卡密激活即可使用
* 还有一些花里胡哨没用的功能
* 支持安卓13画质修改
